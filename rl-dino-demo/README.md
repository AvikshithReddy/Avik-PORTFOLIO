# Dino RL Sandbox

A self-contained, browser-based reinforcement learning demo that teaches Q-learning through an interactive dinosaur story.

## Run
Open `rl-dino-demo/index.html` in a browser.

## What it shows
- Q-learning in a grid world
- Reward shaping (food, lava, rocks, step penalty)
- Exploration vs exploitation (epsilon-greedy)
- Dynamic environment changes
- Value function visualization via cell colors
- Guided step-by-step learning flow

## Controls
- Start/Pause training
- Step once
- Reset
- Tune alpha, gamma, epsilon, step penalty
- Add food, lava, rocks by clicking the grid
- Use Guided Journey to apply each teaching step

## Notes
This is intentionally lightweight and runs entirely on the client.
